Website Indrabath Trans merupakan sebuah wesbite yang digunakan untuk memudahkan user dalam mencari travel dan jasa angkut.
Terdapat beberapa menu pada website ini yaitu home, pelayanan, tarif, kendaraan dan kontak.
Pada saat user menekan tombol pesan sekarang maka user akan otomatis terdirect ke what'sapp Indrabath Trans dan akan melanjutkan proses negosiasi tujuan serta pembayaran melalui What'sapp.
Kemudian terdapat pula fitur login dan logout yang sudah terontetikasi menggunakan MD5 serta fitur register, Apabila user mencoba menerobos masuk kedalam halaman dashboard admin maka user akan diminta untuk melakukan login terlebih dahulu.
Terdapat juga sistem CRUD (Create, Read, Update, Delete) pada bagian gallery yang berguna untuk menambahkan, mengupdate serta menghapus data mobil yang akan ditampilkan pada halaman utama.

Sebelum Menjalankan website, harap import database terlebih dahulu.
